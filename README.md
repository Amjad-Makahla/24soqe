# Car Sales Platform

A web-based car sales platform that allows users to upload and advertise their cars for sale.
All listings require admin approval before being published to ensure quality and reliability.

## Features
- User car listing submission
- Admin approval system for posts
- Image upload support
- Car details (description, price, location, phone number)
- Commenting system
- User account activation and deactivation
- Admin control panel

## Roles
- User: Upload cars, manage listings, comment on posts
- Admin: Approve posts, manage users and platform content

## Tech Stack
- PHP
- MySQL
- HTML / CSS
- JavaScript
